# PersonaSphere

智能人脉管理系统 - 轻松管理你的人际关系网络

### 当前状态：本地Web演示版

#### 一句话简介
一款面向个人与商务人士的“AI增强，人工确认”型智能人脉管理器。通过自然语言输入，自动构建结构化人物档案与可视化社交网络。

---

### 核心工作流（Workflow）
1.  **极简录入**：用户通过打字、语音、图片或文档（自然语言）输入关于某个人的碎片化信息。
2.  **AI抽取**：后端调用大模型（DS）自动提取关键信息，告别繁琐表单。
3.  **分类建档**：将非结构化数据分类整理，形成结构化的人物档案。
4.  **可视化呈现**：生成关系网络图与圈子泡泡图，洞察人脉层级。

---

### 核心功能亮点
-   **智能结构化抽取**
    自动提取并归纳为三类核心信息：
    -   **个人信息**：基础资料、背景经历。
    -   **事件与未来约定**：历史互动、待办事项、约会。
    -   **资源方向**：潜在合作机会、资源对接意向。

-   **可视化洞察**
    -   **关系网络图**：直观展示人与人之间的连接强度与路径。
    -   **圈子泡泡图**：清晰划分社交圈层（如：家人、同事、行业大牛），快速识别核心人脉。

-   **用户掌控与干预（核心体验）**
    -   **AI增强，人工确认**：所有AI生成的内容需经用户审核确认，确保准确性。
    -   **冲突裁决**：遇到信息冲突或歧义时，由用户做最终决定。
    -   **自定义保护**：用户手动填写或修改的内容，永不会被AI自动覆盖，尊重用户意愿。

---

### 核心价值
1.  **极简高效**：用日常对话或文件即可更新档案，极大降低录入门槛，提升移动办公效率。
2.  **结构清晰**：将杂乱的人际情报转化为结构化的可管理资产。
3.  **决策支持**：通过可视化视图，快速发现关键人和潜在资源方向。

## 技术栈

### 后端
- FastAPI - 高性能 Python Web 框架
- SQLite + SQLAlchemy - 数据存储
- Pydantic - 数据验证

### 前端
- React 18 + TypeScript - 现代前端框架
- Ant Design - UI 组件库
- Zustand - 状态管理
- Cytoscape.js - 关系图谱可视化

## 快速开始

### 后端启动

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

然后在浏览器打开 http://localhost:5173
